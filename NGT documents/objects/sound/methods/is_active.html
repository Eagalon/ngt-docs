sound object


This method will determine whether the sound is currently active

bool is_active()

Parameters

None.

Return value

true on If the sound object is currently active, false otherwise

Remarks

this method determines the status whether the sound object is currently active or not.

Example

// Load a sound into memory, play it, and close it. however, we will manually wait for it to finish, and check to see If the sound is loaded properly.
void main()
{
init_engine();
sound test;
test.load("c:\\windows\\media\\ding.wav");
if(!test.is_active())
{
alert("error","the sound could not be actived!");
quit();
}
test.play();
while(test.is_playing())
{
update_game_window();
}
test.close();
}
