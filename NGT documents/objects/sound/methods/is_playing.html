sound object


This method will determine whether the sound is playing

bool is_playing()

Parameters

None.

Return value

true on If the sound object is currently playing, false otherwise

Remarks

this method determines the status whether the sound object is currently playing or not. this is useful in your sound to stop If the specified code is executed.

Example

// Load a sound into memory, play it, and close it. however, we will manually wait for it to finish.
void main()
{
init_engine();
sound test;
test.load("c:\\windows\\media\\ding.wav");
test.play();
while(test.is_playing())
{
update_game_window();
}
test.close();
}
