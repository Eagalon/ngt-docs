datetime object


This method retrieves the month of the date on the current object.

get_month()

Parameters

None.

Return value

The current set month of the date.

Remarks

You can use this method to retrieve the month.

Example

// Displays the date and time.
datetime d;
void main()
{
alert("hello", d.get_month() + ", " + d.get_month() + ", " + d.get_day() + ", " + d.get_hour() + ", " + d.get_minute() + ", and " + d.get_second());
}
