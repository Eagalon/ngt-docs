datetime object


This method retrieves the minute of the date on the current object.

get_minute()

Parameters

None.

Return value

The current set minute of the date.

Remarks

You can use this method to retrieve the minute.

Example

// Displays the date and time.
datetime d;
void main()
{
alert("hello", d.get_minute() + ", " + d.get_month() + ", " + d.get_day() + ", " + d.get_hour() + ", " + d.get_minute() + ", and " + d.get_second());
}
